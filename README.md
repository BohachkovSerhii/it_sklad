# IT Sklad - Система обліку ІТ обладнання

Сучасна система для інвентаризації та обліку ІТ активів підприємства, побудована на Next.js, Prisma та SQLite.

## Основні функції
- **Облік обладнання**: додавання, редагування та списання активів.
- **Рольова модель**: підтримка ролей Admin, Editor та Viewer (NextAuth.js).
- **Штрих-коди**: автоматична та ручна генерація штрих-кодів для картриджів.
- **Транзакції**: історія переміщення обладнання між співробітниками та відділами.
- **Керування користувачами**: адмін-панель для створення та видалення акаунтів.

## Швидкий старт (Windows)
1. Встановіть [Node.js](https://nodejs.org/).
2. Запустіть `start.bat`.
3. Відкрийте [http://localhost:3000](http://localhost:3000).
4. Логін: `admin`, Пароль: `admin`.

## Технології
- **Framework**: Next.js 15+ (App Router)
- **Database**: SQLite (via Prisma)
- **Auth**: NextAuth.js (bcryptjs)
- **UI**: Vanilla CSS (Glassmorphism design)
- **Barcode**: JsBarcode
